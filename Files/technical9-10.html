Overview

The Grades 9-10 Computer Engineering program introduces students to core hardware and software principles. Through subjects like Engineering Drawing, learners master orthographic projections and CAD basics using tools like AutoCAD LT. Computer Fundamentals covers binary/hexadecimal systems, logic gates, and hands-on PC assembly/disassembly labs. This bridges theoretical knowledge with vocational skills relevant to Nepal's growing IT sector.

Students begin C-Programming with structured problem-solving (e.g., creating calculators using loops/functions). Web Page Development (HTML5/CSS3) tasks include building static websites with Nepal-centric themes (e.g., tourism portals for Pokhara). Practical sessions emphasize debugging techniques and W3C validation standards, preparing students for real-world web design challenges.

The curriculum links computing with electronics through Basic Electrical Engineering, where students construct circuits using breadboards and simulate logic gates (AND/OR/NOT). Projects like designing a "Traffic Light Controller" with IC 555 timers demonstrate interdisciplinary applications. Safety protocols for handling soldering irons and multimeters are strictly enforced.

40% of assessment revolves around practical work: Term projects (e.g., coding a student grade calculator in C) Hardware labs (e.g., measuring voltage drops in parallel circuits) Portfolio submissions (website screenshots with responsive design proofs)

Academic Tracks

The Grade 9-10 Computer Engineering curriculum in Nepal, part of the Technical and Vocational stream, aims to equip students with fundamental knowledge and skills in computer science and ICT. This includes digital literacy, problem-solving, and foundational concepts in programming, web development, and computer hardware and software. The curriculum also incorporates practical skills like using electrical equipment and handling engineering curves.

Cirriculum

The technical stream for grades 9-10 emphasizes applied computing skills, blending theory with intensive practical work. Students learn to troubleshoot hardware, write code in C, and develop websites, aligning with Nepal’s demand for IT-skilled graduates 412. Labs simulate real-world scenarios, such as configuring networks or repairing PCs, while projects like creating digital logic circuits reinforce theoretical concepts. Competencies include assembling computers, debugging code, and drafting technical diagrams—preparing students for advanced studies or vocational roles.

Based on the above context, the following competencies have been determined for early childhood development and education:

(A) Competence

Hardware & Networking

  • Assemble/disassemble PC components with ESD precautions.
  • Configure IP addresses and troubleshoot LAN connectivity.
  • Identify logic gates (AND, OR, NOT) using truth tables.

Programming

  • Write C programs with loops/functions (e.g., Fibonacci series).
  • Debug syntax errors using GCC compiler.
  • Develop static webpages (HTML/CSS) meeting W3C standards.

Engineering Fundamentals

  • Draft isometric projections per CDC drawing standards.
  • Calculate resistance in series/parallel circuits.
  • Demonstrate soldering safety protocols.
