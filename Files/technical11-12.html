Overview

Advanced Specialization
The NEB-aligned syllabus delves into Computer Architecture (von Neumann model, pipelining) and Database Management (SQL queries, normalization). Students implement relational databases for local business case studies (e.g., inventory systems for Kathmandu shops).

  • Operating Systems & Networking
    • Hands-on Linux labs teach command-line operations (bash scripting, file permissions).
    • Networking modules cover OSI model layers, with practicals on configuring LANs using CAT6 cables and switches.
    • Students troubleshoot simulated network outages.

Microprocessors & Embedded Systems
Using 8085/8086 kits, learners write assembly language programs (e.g., Fibonacci sequence generators). IoT concepts are introduced through Arduino-based projects (soil moisture sensors for Nepali farms).

  • Industry-Aligned Competencies
    • Developing Java/Python applications with GUI (Tkinter, JavaFX).
    • Configuring Apache web servers on Ubuntu.
    • Presenting technical documentation per NEB standards.
  • University Readiness
    • The program aligns with IOE entrance syllabi, emphasizing:
      • Algorithmic problem-solving (Big-O notation).
      • Hardware-software integration projects.
      • Research papers on emerging tech (AI/blockchain applications in Nepal).

Academic Tracks

The CDC (Curriculum Development Centre) curriculum for Class 11-12 Computer Engineering in Nepal focuses on developing foundational computer science knowledge and skills. This includes understanding computer systems, number systems, Boolean logic, programming concepts, and web development. The curriculum also emphasizes developing soft skills like critical thinking and creativity.


Subject Description
Computer Hardware & Architecture Processors, memory hierarchy, I/O systems.
Operating Systems & Networking Linux commands, LAN/WAN setup.
Database Management SQL, relational models.
Object-Oriented Programming (OOP) Java/Python basics.
Microprocessors Assembly language, embedded systems.

Key Focus Areas

  • Advanced labs (e.g., setting up servers, OOP projects).
  • Industry-aligned certifications (e.g., hardware troubleshooting).

Cirriculum

The Class 11-12 curriculum delves into specialized IT domains, mirroring diploma-level coursework. Students dissect computer architecture, design databases, and build networked systems, with 40% of time allocated to practicals 9. Projects like developing a student management system in Java or configuring routers provide hands-on experience. Graduates emerge with skills akin to a Diploma in Computer Engineering, qualifying them for IT apprenticeships or further studies

Based on the above context, the following competencies have been determined for early childhood development and education:

(A) Competence

Advanced Systems

  • Explain von Neumann architecture with cache memory diagrams.
  • Implement SQL queries (JOIN, GROUP BY) for inventory databases.
  • Write 8086 assembly code for arithmetic operations.

Software Development

  • Build Java GUI applications (AWT/Swing).
  • Design normalized ER diagrams for school management systems.
  • Configure Apache web servers on Linux VM.

Industry Readiness

  • Present technical reports per NEB formatting guidelines.
  • Develop Arduino prototypes with sensor integration.
  • Solve IOE entrance-level algorithm problems.
